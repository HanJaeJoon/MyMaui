1. 지난 번 논의에서 2안은 가능할듯
  - 내장 카메라 이용
  - 서버에서 프린트 연동
2. 앱 + 서버 형태여야 할듯
  - 디바이스 저장이 불가능함(카카오톡 사진 전송 느낌)
  - 디바이스에 고객 사진 저장(개인정보 문제)
  - 프린트와 연결된 서버로 spool 기능을 이용하면 프린트 연동될듯
  - 그렇다면 서버 - 카메라 연동을 연구하는 방안도 있을 수도 있음
3. 카메라 구매 전 동작 확인을 하고 싶음
